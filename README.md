# DevelopersHub AI/ML Engineering Internship Tasks

## Task 3: Multimodal ML – Housing Price Prediction Using Images + Tabular Data

### Task Objective

Learn how to build a multimodal machine learning model by combining image data and structured tabular data to perform regression.

### Dataset Used

Source: CIFAR-10 (for images) + Synthetic tabular dataset

Features Used: Synthetic features such as bedrooms, bathrooms, and area along with image features extracted using CNN

Target Variable: Simulated house price

### Models Applied

Convolutional Neural Network (ResNet50 for feature extraction)
Neural Network (Multimodal regression model)

### Key Results and Findings

Image data was successfully processed and features were extracted using a pretrained CNN.
Tabular data was generated and combined with image features.
The multimodal model was able to learn from both data types simultaneously.
Performance was evaluated using MAE and RMSE.
Predictions are not highly accurate due to the synthetic nature of the dataset.
The project effectively demonstrates the concept and implementation of multimodal machine learning.
